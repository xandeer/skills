# Weekly Review

## Objective

Run a short weekly review with consistent structure and actionable output.

## Workflow

1. Summarize what was completed this week.
2. List unfinished or blocked work with reasons.
3. Capture 3 lessons learned.
4. Define top 3 priorities for next week.
5. Pick one risk and one mitigation for next week.
